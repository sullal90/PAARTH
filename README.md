

# PAARTH — AI Recitation Companion

![System Architecture](architecture/Architecture.png)

**PAARTH — Pronunciation Analysis and Assisted Recitation Teaching Helper**


**PAARTH** is a real-time AI recitation tutor powered by Gemini.
Users chant a Bhagavad Gita verse, the agent evaluates pronunciation and provides corrective guidance like a traditional guru.
The system uses speech recognition, Gemini reasoning, and voice feedback to create a conversational learning experience.

### 🧠 Meaning Behind the Name
Paarth represents the learner.
System acts like a digital Krishna that:
- listens to recitation 🎤
- corrects pronunciation 📖
- explains meaning 🧠
- guides spiritual learning ✨

Just like Krishna guiding Arjuna in the Gita.

PAARTH combines browser speech input, a Gemini-powered tutoring agent, verse retrieval, and voice-guided feedback to create an interactive Bhagavad Gita recitation coach.
